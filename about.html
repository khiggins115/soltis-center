Howdy!

This site functions as a small web app for you and anyone you know to visualize the data we have collected for from our work with the Soltis Center in Costa Rica

This site has been configured using the Leaflet Javascript library and the data being served to the map is hosted with Esri's ArcServer.

If you're interested in collaborating to this dataset, or editing the data that is currently available, you can contact the Texas A M university Maps and GIS Library. If you'd like to contribute to the Github, you can create a free account and submit a 'pull request' for your submission to be considered for upload.

There is additional data available for download on the 'Data' page of this website
